Student Performance Prediction System

This project is developed by MCA – Section B, Group 1. The system predicts student performance using Machine Learning techniques.

Members: Deepak Singh, Prem Kumar

  • Student input form creation
  • Result display page development
  • Form validation using JavaScript
  • Integration of charts

  • Decision Tree
  • Random Forest
